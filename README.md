# Octy.ai api-micro-services

Python (FastAPI) microservices — 12 API services + 5 background workers,
behind Traefik, backed by MongoDB/Redis/RabbitMQ/S3/SageMaker.

- **Deploying or developing against this codebase?** See [`DEPLOYMENT.md`](DEPLOYMENT.md) —
  architecture overview, local dev setup, and the full production runbook.
- **Rust/WASM rewrite** of this same system lives in [`rust/`](rust/README.md).
